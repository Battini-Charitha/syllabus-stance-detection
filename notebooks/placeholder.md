# Notebooks

This folder contains all Jupyter notebooks for the syllabus stance detection pipeline.

| Notebook | Description |
|----------|-------------|
| 0_data_preparation | Data loading and filtering |
| 01_baseline_models | Classical ML baselines |
| 02_hyperparameter_tuning | Grid search and tuning |
| 03_scibert_finetuning_clean | RoBERTa fine-tuning |
| 04_inter_annotator_agreement | Annotation validation |
| 05_error_analysis | Error analysis |
| 06_data_augmentation | Llama-3 synthetic augmentation |
| 07_large_scale_classification | Two-stage pipeline: Stage 1 : policy relevance filtering using BART-large-mnli (run on CRC cluster); Stage 2 : stance classification using fine-tuned RoBERTa (run on Colab T4 GPU) |
| 08_genai_analysis | GenAI comparative analysis |
